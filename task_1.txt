Мной был выбран интернет-магазин в виде проекта.
Проект будет реализовать имитацию работы интернет-магазина sportmaster.
То есть любой пользователь(зарегистрированный) может оформить заказ
на интересующий его товар по определеннм категориям.
Так же может поместить его в корзину для следующего оформления оформления. 

Проанализировав сайт, я выделил следующую структуру данных:
- Таблица users (пользователи);
- таблица baskets (корзины пользователей);
- таблица orders (заказы);
- таблица orders_products (заказы/продукты);
- таблица catalogs (каталоги);
- таблица products (товары);
- таблица album_images (альбом товара);
- таблица images (фотографии всех товаров);

Описание сущностей:
users - Наши зарегистрированные пользователи которые могут
купить товар;
baskets - корзины пользователей на некий понравившийся товар;
orders - номера заказов которые оформили users;
orders_products - продукт который соответствует номеру заказа;
catalogs - cписок каталогов для систематизации товаров;
products - список товаров доступные в магазине;
album_images - у каждого товара должен быть альбом с фотографиями;
images - хранит фотки для каждого альбома товара.
